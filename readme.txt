== Description ==

By this smart plugin you can protect your login page from Brute-force attacks also you can track login history.

Features:
1. Captcha on the admin login page to protect against auto-hacking.
2. You can set the number of failed logins to show Captcha or showing it always.
3. Login Blocker option to block the login processe for a period of time after specific number of failed logins.
4. Login history to show all login processes in details.

You can see more about this plugin at <a target="_blank" href="http://www.clogica.com/wordpress-login-security-and-history-wordpress-plugin.htm">
WP Login Security and History WordPress Plugin</a>

== Installation ==

1.  Go to the wordpress control panel and go to “Plugins” menu and press the button “Add New”
2.  Go to the tab “Upload” and choose the file “wp-login-security-and-history.zip”, click the button “Upload” to get the file uploaded.
3.  Lockup the plugin, and click “Activate” to get it active.
4.  Go to the Options page and configure the plugin.
